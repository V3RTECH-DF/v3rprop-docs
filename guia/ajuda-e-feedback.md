# Ajuda e Feedback

No cabeçalho de **qualquer tela** do V3RProp, no canto superior direito, ficam dois botões sempre à mão:

- **Manual do Usuário** — abre este manual em uma nova aba.
- **Enviar Feedback** — abre um formulário para você mandar sugestões, dúvidas ou relatos de problema direto para a equipe V3RTECH.

## Enviar um feedback

1. Clique em **Enviar Feedback** no cabeçalho.
2. Escolha o **tipo**: Sugestão, Dúvida ou Bug.
3. Escreva sua **mensagem** (é o único campo obrigatório).
4. Confira o **e-mail para resposta** — já vem preenchido com o e-mail comercial das suas Configurações, mas você pode trocar.
5. Clique em **Enviar**.

![Formulário de feedback do V3RProp preenchido](/assets/screenshots/manual-feedback.png)

Você recebe uma **cópia por e-mail** do que enviou, e os últimos envios ficam listados no rodapé do próprio formulário.

## O que é enviado

Para agilizar o suporte, todo feedback acompanha um **diagnóstico técnico** — sem dados pessoais: a versão do plugin, do WordPress e do PHP, o idioma, o tema ativo, o número de plugins ativos e um resumo da sua configuração (se Gotenberg, chave de API e certificado estão configurados — apenas "sim/não", nunca os valores).

Se você marcar a caixa **"Incluir dados da minha organização"**, o envio passa a incluir também o nome, o CNPJ e a URL do seu site — útil para a equipe identificar sua conta.

{: .note }
> **Seus segredos nunca são enviados**
>
> Senhas, o certificado digital, o valor da chave de API e dados de clientes/propostas **nunca** acompanham o feedback. Use o link **"O que será enviado"** dentro do formulário para ver o resumo antes de mandar.

{: .tip }
> **Bug? Capriche na descrição**
>
> Ao relatar um problema, conte o que você fez, o que esperava e o que aconteceu. Com o diagnóstico técnico que já vai junto, isso costuma ser suficiente para a equipe reproduzir e corrigir.
